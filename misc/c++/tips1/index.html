# 类型擦除

类型擦除是 C++ 中一种用于实现多态性的编程技术，它允许在不牺牲性能或引入不必要的运行时开销的情况下进行多态性操作。 通过隐藏对象的实际类型并提供统一的接口，类型擦除使得可以以多态的方式处理不同类型的对象，同时在运行时推迟对实际类型的确定。
